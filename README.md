# Teste GitHub CLI

Este é um repositório de teste para demonstrar as capacidades do GitHub CLI.

## Funcionalidades Demonstradas

- Criação de repositórios
- Clonagem de repositórios
- Gerenciamento de issues
- Criação de pull requests
- Busca de repositórios
- Visualização de informações de usuário

## Comandos Úteis

```bash
# Listar repositórios
gh repo list

# Criar um novo repositório
gh repo create nome-do-repo --public

# Criar uma issue
gh issue create --title "Título" --body "Descrição"

# Criar um pull request
gh pr create --title "Título" --body "Descrição"
```

## Recursos Avançados

O GitHub CLI oferece integração completa com a API do GitHub, permitindo automação de workflows e operações complexas diretamente da linha de comando.
